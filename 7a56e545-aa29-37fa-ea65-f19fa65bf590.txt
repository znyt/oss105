　　灭世仙君是真的，快要被叶尘气吐血了！
<br/>　　原本，
<br/>　　在他看来，以叶尘的天赋，倘若能够继承他的传承，千年之内，必定可以追上云苍生的脚步，说不定自己的大仇，到时候也就可以报了。
<br/>　　可是，他怎么也没有想到，叶尘居然拒绝拜他为师！
<br/>　　不过，随后叶尘所表现出来的强大实力，又让灭世仙君感觉到了希望，所以才提出，让叶尘立下本命誓言，为自己报仇。
<br/>　　至于时间，对于他们这些活了无数岁月的修真者来说，一千年、两千年，和世俗界普通人的一两年差不多，所以他根本就没有打算，对叶尘做出时间限制。
<br/>　　可是现在呢，叶尘居然自己提出，要在百年之内，杀了云苍生，还要灭了无极宗！
<br/>　　你以为云苍生就是这么好杀的？无极宗就是那么容易灭的吗？
<br/>　　简直就是不知死活！不知所谓！不知道天高地厚啊！
<br/>　　而叶尘，却依旧一脸淡然，脸上挂着淡淡的笑容，仿佛只是做了一件微不足道的小事一般。
<br/>　　“前辈，我已经立下本命誓言，这灭世魔刀是不是就属于我了呢？”
<br/>　　“你你你……”
<br/>　　灭世仙君指着叶尘半天，愣是一句话也说不出来，如果他不是自己清楚，以自己如今的状态，根本奈何不了叶尘，恐怕非得将他抓住暴打一顿不可。
<br/>　　“唉！”
<br/>　　灭世仙君郁闷了半天，最终重重叹了一口气，道：
<br/>　　“罢了罢了！我的这个大仇，怕是永远都报不了啦！你自己好自为之吧！”
<br/>　　说完这话，灭世仙君随手一挥，那把灭世魔刀，便自动飞到了叶尘的跟前。
<br/>　　叶尘顿时大喜过望，伸手一抓，握住灭世魔刀的刀柄，
<br/>　　轰！
<br/>　　一股强大令人心悸的毁灭气息，顿时并顺着刀柄，灌入叶尘的身体之中。
<br/>　　“嗯？”
<br/>　　叶尘感应了片刻，忽然眉头微微一皱，不由得失声道：
<br/>　　“这把刀，怎么会有一股不容于这个世界规则的混沌之力？”
<br/>　　灭世天君听到叶尘这话，苍老的脸上，先是露出一抹惊色，继而笑道：
<br/>　　“好小子啊！没想到你竟然连这个都能感应的出来，那我就实话告诉你吧！本座的这件法宝，不是仙器，也不是神器，它乃是一件真正的混沌灵宝！”
<br/>　　“什么！！”
<br/>　　叶尘听到这话，顿时两眼一瞪，
<br/>　　“这灭世魔刀，竟然是一件混沌灵宝？”
<br/>　　要知道，混沌灵宝，诞生于宇宙外的混沌之中，向来只是传说中的存在，稀有程度甚至比神器还有过之而无不及！
<br/>　　当然了，这并不是说，混沌灵宝就一定比神器强大。
<br/>　　很多混沌灵宝，和世界规则之力格格不入，会被世界之力限制，甚至有些灵宝的威力，还不如仙器有用。
<br/>　　不过据说，混沌灵宝若是在宇宙之外，无尽混沌之中，就能发挥出最强的威力，连神器也无法与之相比！
<br/>　　“本座的这件法宝，的确是混沌灵宝，不过却是一件残品，所以大约相当于极品仙器的层次！不过……”
<br/>　　灭世仙君说到这里，微微一顿，接着又道：
<br/>　　“小子，还记得之前，本座曾经说过，这件法宝之中，隐藏着一个天大的秘密吗？”
<br/>　　叶尘顿时两眼一亮，连忙追问道：
<br/>　　“什么天大的秘密？”
<br/>　　灭世仙君深吸了一口气，方才幽幽开口道：
<br/>　　“当初，本座在一处绝地之中，意外得到这件法宝，当时它实在太破了，本座一开始并没有将它放在心上，便随手丢进了乾坤镯之中，和其他法宝放在了一起……”
<br/>　　“没想到，奇怪的事情发生了，这把刀竟然会自动吸收其他法宝的灵性，和它放在一起的那些法宝，时间一久，就全部变成了一堆废铁！”
<br/>　　叶尘听到这里，也不由得微微一惊，
<br/>　　“还有这等奇事？”
<br/>　　灭世仙君点了点头，道：
<br/>　　“我当时也吓了一跳，不过也激起了我的好奇心，于是我便开始搜集各种低级法宝，以供它吞噬！”
<br/>　　“没想到随着吞噬的法宝越来越多，这把刀的威力也越来越强，再吞噬了数以万计的低级法宝之后，它居然一举进阶到仙器的层次！”
<br/>　　“一开始，我真的高兴坏了，要知道，仙器可不是大白菜，哪怕是八大仙门之中，也没有多少，而我当时才不过元婴境的修为，能拥有一件仙器，足以让我的战斗力，跻身年轻一代中的顶级行列！”
<br/>　　“更何况，只要不停的吞噬法宝，这把怪刀的等级就会继续提升，甚至成为神器都不在话下！”
<br/>　　“就在我志得意满之时，却浑然没有发现，随着这把刀使用的次数越来越多，我的心智竟然也不知不觉受到了影响，我开始变得嗜杀成性，甚至已经开始有些无法控制自己！”
<br/>　　“到了这个时候，我才发现，这竟然是一把能够影响人心的魔刀！”
<br/>　　听到这里，叶尘顿时眉头一皱，
<br/>　　倘若真如灭世仙君所说，这把刀可以影响人的心智，那还真是一件邪物，以后恐怕还要尽量少用才是！
<br/>　　灭世仙君继续又道：
<br/>　　“等我反应过来之后，我也曾经想过，将这把魔刀封印起来，可是面对变强的诱惑，以及仇人的追杀，我终究还是无法下定决心。”
<br/>　　“再后来，随着我杀的人越来越多，凶名也越来越盛，仇人自然也越来越多！”
<br/>　　“没办法，我只好找到当时，我最好朋友云苍生帮忙！”
<br/>　　“在云苍生的帮助之下，我成为了无极宗的客卿长老，有了无极宗这个大靠山，之前的那些仇人，再想要找我报仇，自然就要掂量一二了！”
<br/>　　“而作为报答，我将魔刀的秘密，也全都告诉了云苍生，并且邀请他一起研究这把魔刀的秘密，云苍生很爽快的答应了，并且信誓旦旦的说，一定帮我驱除心魔……”
<br/>　　灭世仙君说到这里，眼眸中忽然泛起一抹浓浓的恨意，声音也变得有些冰冷起来，
<br/>　　“只是我怎么也没有想到，原来从一开始，云苍生就刻意与我结交，到后来我被人追杀，也全都是他一手设计，为的就是我手上的魔刀，还有魔刀背后的秘密！”
<br/>(本章完)
