　　第725章
<br/>　　动手的本事刘发财不如何，但开口忽悠人的本事，却很少有人能和他相提并论。他一边往嘴里填东西，一边胡吹海侃，哄得陈德山一愣一愣的。
<br/>　　他险些都要相信了，越看程大雷这一行人越不简单。程大雷身上一股杀伐气，刘发财也能言善道得很，那眼中闪烁着智慧的老叟，貌美无双的女子，沉默寡言的黑衣尼姑……明显不是普通的团伙。
<br/>　　他将六手金佛重新推到程大雷面前，道：“牛将军，万万不可再推辞，请一定要手下，让老朽略表心意。”
<br/>　　程大雷一把推开，晃了晃肩膀站起身：“倦了，回去歇着。”
<br/>　　刘发财还准备说什么，程大雷已踉跄走出屋门。陈德山急忙忙让人在前面引路，担心程大雷因醉酒倒在半路上。
<br/>　　等几人都离开后，房间内只剩下陈德山与陈忠两人。陈德山在这个时候才慢慢安静下来。
<br/>　　陈忠垂手立在一旁，皱眉思索着什么。
<br/>　　“老爷，还是看不出他们的来路？”
<br/>　　陈德山点点头：“应该是军伍中人不假，身上那份杀伐气是千锤百炼出来的。但只是一个小小的边关守军么，刚才他起身时，其他几人都停下筷子，很显然对他的命令言听计从。究竟是哪条路来的神仙，我这脑子里当真对不上号。”
<br/>　　“老爷也不必多想，看来他们对陈家庄没有恶意，这就够了。”
<br/>　　“是呐，你说得有道理，没有恶意就好。”陈德山又摇摇头：“可这扑面而来的善意，实在像米饭里的沙子，让人心里不太舒服。”
<br/>　　“老爷，那冯四方如何处置？”
<br/>　　冯四方负伤之后，腿脚行动不便，就被陈德山手下家丁擒住，如今丢在后院柴房中。
<br/>　　“这等畜生，活着也是恶心我。”陈德山眉头皱起，眉宇间杀机尽现：“今夜就送他上路。”
<br/>　　“是。”
<br/>　　陈家庄的位置，距离附近城池都有些距离，属于天子王法管不着的地方。碰到公媳扒灰，叔嫂通奸这种事，他也要带行王法，送人上路这种事他也没少做过。当然，具体是浸猪笼，还是麻袋装好活埋，那就不一定了。
<br/>　　到得次日天明，陈德山便早早起床，去往程大雷所居住的院落。
<br/>　　“牛将军，牛将军……”
<br/>　　院门推开，已经人去房空，院内的马匹兵器也同时不见了。
<br/>　　陈德山无可奈何的摇摇头，发出一声长叹：“高攀不起呐。”
<br/>　　程大雷明显不愿和陈家庄扯上什么关系，一饭之恩就只是一饭之恩。如陈德山这种人，早已活成了人精，与他交往过多，说不得等你与他掏心掏肺的时候，他心里冷冷哼一声：蠢货。
<br/>　　于是程大雷天不亮就起身，悄无声息的离开了陈家庄。一行五人重新踏上赴京的路。
<br/>　　总算是好好休息一夜，路上大家都有了精神。
<br/>　　“也不知道那陈家小姑娘生得如何模样，咱们也没看看，还是走得太匆忙。”徐神机扶须道。
<br/>　　“嗳，再好看能好看到什么程度，总之是比不上崔姑娘。”刘发财道。
<br/>　　马车里窝着的崔白玉听到这话，心中或多或少有些高兴。爱美之心人皆有之，她也觉得这刘发财比较会说话。
<br/>　　徐神机点点头：“说得也有道理。”
<br/>　　“我看未必，不能坐进观天，小觑了天下，这方水土说不得也能生养美人。”
<br/>　　然后，程大雷不顺耳的声音传过来，崔白玉心底重重冷哼一声。生得如程大雷这般模样，就该找块石头直接撞死，竟然还好意思评点他人美丑。
<br/>　　一行人继续东行，在马背上胡乱说些闲话。路程自然谈不上愉悦，有些道路经年无人走，已经长满荆棘荒草，几人不得不下马劈开荆棘开路。
<br/>　　天气依旧炎热，几人不多时便汗流浃背，程大雷三个还受得住，可这却苦了马车里的崔白玉。香腮流粉汗，她轻轻挥着玉手，但也卷不起多少风。
<br/>　　“呔！”
<br/>　　这时候从林子里突然跳出来两个壮汉，突然拦在路上。一胖一瘦，胖得浑圆，瘦得见骨。
<br/>　　“此山是我开，此树是我栽。”
<br/>　　“要想从此过，留下买路财。”
<br/>　　二人手里攥着朴刀，跳出来便开口大喝唱起了山歌。
<br/>　　程大雷和徐神机都抿嘴笑了，没来由的这两个蟊贼给他们一种亲切感。
<br/>　　“喂，怎么称呼，报个腕儿。”
<br/>　　两蟊贼面面相觑，也劫了不少人，但像程大雷一伙这么镇定自若的还很少见。
<br/>　　“钻地鼠林英。”
<br/>　　“翻江鼠林雄。”
<br/>　　程大雷点点头，嘴里喃喃道：“倒是个好名字，只是这外号叫起来显得一点也不威风。”
<br/>　　二鼠对视一眼，瘦子林英道：“打劫呢，打劫呢，搞搞清楚好不好。”
<br/>　　林英感觉自己的职业收到侮辱，他大手一挥道：“都出来。”
<br/>　　哗啦啦从林子里又窜出三个人，有人拿大棒，有人拿铁棍，身上穿着破布麻衣，看起来奇形怪状的。
<br/>　　“好啦，好啦。”程大雷摆摆手：“碰上我是你们的福分，赶紧帮忙给清一清路，然后我就放你们一条路。”
<br/>　　五只山贼都感觉不可思议，林英有些气急败坏：“你想什么呢，我们在打劫好不好？”
<br/>　　程大雷冲刘发财点点头：“收拾一下，我们也省些力气。”
<br/>　　刘发财晃了晃肩膀，脸上挂着笑容：“好小子，来来来，今天让你们认识认识祖宗。”
<br/>　　林英大吼一声，拎着刀向刘发财扑过来。刘发财如今在蛤蟆寨已经显不出武艺，但面对几个小蟊贼还是有几分高人风范。举手投足间，便将五个贼人料理了，他们倒在地上，抱着脑袋，身上或多或少都挂了彩。
<br/>　　五只贼这才知道碰见硬点子，林雄眼珠钻了一圈，指着程大雷鼻子道：“你有胆子就放我们回去，等我叫齐弟兄再堂堂正正一战。”
<br/>　　“好啦，别卖机灵了。”程大雷道：“都拿起兵器，帮我们开路，到时候留不留你们性命，且看你们活干得如何。”
<br/>　　五贼对视一眼，无可奈何的撅起屁股，在前面替程大雷一行扫清路上的障碍。
<br/>(本章完)
