　　第104章
<br/>　　“十”
<br/>　　“九”
<br/>　　“八”
<br/>　　……
<br/>　　高飞虎数得很慢，但却给黄三元带来巨大的压迫感，反观身边的关鱼，却仍旧一张重枣脸，眼睛眯得一条线。
<br/>　　哥呀，你究竟有什么本事，使出来好不好。如果没有本事，请不要摆出高手的表情。
<br/>　　远远山道之上，程大雷奚落高飞豹没取到应有的效果，转身开始奚落小白狼。
<br/>　　“白当家，你看你这山贼做得，竟差点也被人劫了。就好像你去黑石城抢劫，如果马被人偷走，是不是很丢人。”
<br/>　　一路行来，程大雷这样的风凉话说了不少，很是拉了一波仇恨。
<br/>　　正行着，前方乌压压黑影，程大雷停下马，表情有些发愣。
<br/>　　“你们干嘛呢？”
<br/>　　哗！
<br/>　　人群忽然扭过头来，看到程大雷时愣住。高飞虎正数着数，猛回头也是吓了一跳。
<br/>　　这煞星怎么刚好这个时候回来了！
<br/>　　嘟，收到恐惧值99
<br/>　　嘟，收到恐惧值88
<br/>　　嘟，收到恐惧值……
<br/>　　程大雷有些无语，心想自己还什么都没做呢，这帮人至于这么害怕嘛。
<br/>　　“大当家，他们要抢我们山寨！”黄三元用最大的声音喊道。
<br/>　　呼啦！
<br/>　　青牛山的山贼转圈将程大雷围住。
<br/>　　程大雷委实也吓了一跳，若是一家一户想要抢自己，这也不算奇怪。可眼前足有三五千人，整个青牛山的山寨都出动了。关键是为什么呢，自己有那么招人恨么？
<br/>　　不仅是程大雷，高飞豹和小白狼也有些困惑，小白狼分明看到自己杏花岭的人也加入这次抢劫蛤蟆寨的行动中。
<br/>　　为啥呀？
<br/>　　“秦蛮，子龙动手！”程大雷爆喝。
<br/>　　在高飞豹和小白狼还没回过神时，二人已被踹到马下，再回过神已经分别有两柄枪抵在二人脖子上。
<br/>　　“程当家，咱们是自己人，不必如此吧？”小白狼苦笑道。
<br/>　　“别废话，好像你没抢我山寨似的。”程大雷没给她好脸色看，冲着前方大声喊道：“高当家，近前说话，大家向来和睦，为何要对我蛤蟆寨动手？”
<br/>　　“程寨主勿怪，弟兄们只是混口饭吃。”
<br/>　　“你想混口饭吃，就要抢我山寨。”程大雷冷笑道：“大家也别废话，你兄弟在我手里，想要他活命，带着你的人滚，我当什么事情都没有发生。如若不然，我一枪捅了他，咱们放手打一场，说实话你的人虽多，我蛤蟆寨还真没有怕的道理。”
<br/>　　程大雷五个人的时候就敢挡黑石城一千，何况现在不仅有了人，还多了赵子龙这种猛将，他更加不会害怕。
<br/>　　高飞虎也没想到，偏巧不巧，自己弟弟竟落在对方手中。现在所有人都看向他，等着他做最后的决断。
<br/>　　于虬然近前低声道：“大当家，量小非君子，无毒不丈夫，事到如今，咱也只能狠下一条心来。”
<br/>　　高飞虎斜睨了他一眼：“狠你麻痹，那是我亲弟弟！”
<br/>　　高飞虎没有杀弟的决心，高飞豹也没有牺牲自己的觉悟，形势僵持下来。
<br/>　　“看来高寨主还没拿定主意，那我就帮你一下。”程大雷伸手一指：“谁去为我取他项上人头！”
<br/>　　张肥喳喳笑道：“吾万军丛中取上将人头如探囊取物。”
<br/>　　“哎呦呦。”程大雷赶忙将这愣货拦住：“三爷，冷静，冷静。”
<br/>　　“阁下的首级，我收下了。”
<br/>　　此刻，忽然一白袍小将冲了出去。银枪白袍，宛若一道白色闪电，眨眼之间已冲进阵中，硬生生闯出一道线。
<br/>　　硝烟战场上的一点银芒，千军万马中的一条游龙，正是银枪龙胆的常山赵子龙。
<br/>　　只是一合，便将高飞虎挑落马下，然后扬长而归，如若无人之境。
<br/>　　“大哥！”高飞豹大喊，他却以为高飞虎已被斩杀。
<br/>　　“没事，我没事！”
<br/>　　高飞虎从地上爬起来，到现在他没弄明白怎么会事，只是肩膀已经出现一个血洞。
<br/>　　人人目裂胆寒，天下竟还有如此骁勇之将，偏偏还落在程大雷手中。有此一人在，便己方有千军万马又如何。
<br/>　　高飞虎背后的冷汗一层又一层，他刚才着实在鬼门关前走了一着。
<br/>　　“多谢程寨主饶命。”高飞虎道：“请上山！”
<br/>　　人群分开一条道路，虽然人人都拿着兵器，眼中不乏贪婪，却是无人敢动手，只能目送着程大雷等押着高飞豹和小白狼等人上山。
<br/>　　“还请程寨主放了我兄弟。”高飞虎在山下喊。
<br/>　　“高寨主，我先留你兄弟住几天，改天一定给你送回去。”
<br/>　　“你！”
<br/>　　“怎么高寨主也想来我山上做客？”程大雷冷笑道。
<br/>　　高飞虎一惊，咬着牙道：“程寨主若想留我弟弟，那就让他在蛤蟆寨住几天，只愿他没有什么闪失，否则，也莫怪我鱼死网破。”
<br/>　　“鱼死了网也未必会破。”
<br/>　　程大雷丢下一句话，押着高飞豹和小白狼两拨人走入山寨。回到山寨后，就把他们丢进地牢中。
<br/>　　然后，程大雷把黄三元，关鱼等人召集到议事厅。
<br/>　　“大当家，幸亏你及时回来，现在他们零零散散都撤了。”
<br/>　　黄三元说了一句，看到程大雷还阴沉着一张脸，难道这还不值得高兴。不知怎地会事，黄三元感觉程大雷这次回来和以前有些不太一样，莫名让人觉得怕。
<br/>　　“说说吧。”
<br/>　　“说什么？”黄三元一头雾水。
<br/>　　“呦，还不说实话是不是，非要我问出来么。”程大雷道：“说说这些天你们谁干了什么事，把他们都得罪了？”
<br/>　　“没有啊，我这些天连山寨大门都没有出。”黄三元。
<br/>　　“你也没有？”程大雷问关鱼。
<br/>　　关鱼冷哼一声，没有回答。
<br/>　　看来二人真没有做什么坏事，那程大雷就有些想不明白了。
<br/>　　青牛山五十多家山寨联合，老幼妇孺齐上阵，一共五千多人要打自己的蛤蟆寨。这种劲头就是挖人祖坟也不至于吧……可自己的确没做过什么得罪他们的事，自己的人缘不是一直都好到爆么。
<br/>　　赵子龙和关鱼见面，二人相谈甚欢，一叙别情。
<br/>　　程大雷窝在太师椅上，脑袋里还想不明白这个问题。
