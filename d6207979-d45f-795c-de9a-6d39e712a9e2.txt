<br/>    月上柳梢头，陈明在熹微的烛火下看着黑蟒的奏报，那是关于金毛吼的消息，“你的意思是说，那头老狮子正在四处挑拨我燕山与诸位大妖王的关系，如今诸位大妖王都已经在暗地里和我燕山决裂？”
<br/>    
<br/>        黑蟒拱手道：“不错，数位大妖王都已经暗地中和燕山决裂了，如果不是燕山镖局的存在，燕山的那些妖王炼制的丹药，都快被抢光了。”
<br/>    
<br/>        陈明思量片刻，然后道：“说实话，我对于金毛吼也没有什么好感。既然他想玩出个花样，我们就看看他到底有什么花样。”

<br/>        那头老狮子可是有着lv2的存在，这样的家伙，放在自己家的地盘之中，陈明也不放心，总会搞事情。

<br/>        正好这头老狮子不安分了，干脆先把这几家大妖王全端了算了，反正自己和妖族大君对战之时，估计这些家伙也不安分，会在背后搞事情。

<br/>        陈明看向身前的黑蟒，黑虎，火燕三人，过了七天，本来就只差最后一步，现在已经是有了大妖王的实力，加上银翼和卓青瑶，现在的燕山，可谓是有了五位大妖王战力。
<br/>    
<br/>        十万血衣剑卫，随时枕戈待旦，而且，所有的血衣剑卫已经完成了换装，他们现在使用的杀王剑，可是陈明改进之后的二代产品，灵甲同样也是二代产品。

<br/>        陈明走到地图之前，提笔在地图之上各位大妖王的领地上划了一个叉，“既然他们不想活，那也就不用留他们了。”
<br/>    
<br/>        黑蟒，黑虎，火燕三人看向那地图上多出的七个叉，心头也是一阵火热，黑虎拱手道：“只要山主一声令下，我立马发兵踏平整个十万大山，寸草不生！”
<br/>    
<br/>        “叮，随着实力的日渐增强，只是燕山的地盘已经不能满足燕山的继续扩张，请扫清所有不臣，一统妖域。奖励：未知。”
<br/>    
<br/>        嗯，任务也来了，奖励未知的一般都有着好东西，最好再来个光环奖励啥的。
<br/>    
<br/>        陈明道：“这一次远征，横跨整个十万大山，纵横数万里，这一次远征，我命名为妖族的远征，势要扫清所有不臣，一统妖域！”
<br/>    
<br/>        黑蟒，黑虎，火燕三人此刻也是热血沸腾，作为一位妖王，能够参与如此大事，让他们感到心血激涌，“我等，愿为山主扫清一切不臣！”
<br/>    
<br/>        就在这时，银翼忽然来到此处，朝着陈明禀报道：“启禀山主，外面有着一位自称紫霞的大君，带着数百人王，前来拜访山主，说是要助山主一臂之力。”
<br/>    
<br/>        陈明的眉毛跳了跳，“他们来这儿干嘛？”
<br/>    
<br/>        陈明继续道：“算了，既然来都来了，让他们在沙海之外我们出兵的经过的地方等候吧，好生招待一番，不要怠慢了，这些小家伙，都是古仙门的天才。”
<br/>    
<br/>        银翼愣了愣，山主现在这么大的面子吗？古仙门都得来巴结他？不过看到陈明排面越大，他也是越高兴，出去招待去了。
<br/>    
<br/>        陈明道：“明日清晨，在沙海之内的边缘，寻找一处誓师，燕山出兵，开始远征！”

<br/>        在夜里，陈明召集了众多的记名弟子以及燕山上百位被夺了兵权的妖王在一处，三千六百记名弟子汇聚在一处，妖王们在后面观看，陈明看向众多的记名弟子，他们已经乖乖盘膝坐好，其实严格算起来，他们都还是少年，成年的并不多，不过在修行界来说，年少高位的多得是。

<br/>    
<br/>        “开拓令如下：凡是有资格接受开拓令的记名弟子，皆为燕山归属下的领主，所有领主，开拓令期间，拥有无限制组建自己的血衣剑卫的权力，兵书《血衣剑卫》全面开放，不得外传，拥有无限制炼丹房的权力，炼丹房不得外传，你们每个人都可以得到一份地图，凡是地图上敌对的目标，都是你们可以进攻占领领土的目标。”
<br/>    
<br/>        “所有接受开拓令的弟子，可以凭借开拓令在燕山钱庄获得十万灵石的基础战争款，所有领主，必须上交每年获得的十分之一资源，作为学费，开拓令期间，所有税收减免。”

<br/>        这些话听得记名弟子激动不已，我勒个去，师父这是要干嘛？血衣剑卫和炼丹房全面解封，那将是怎么样的一股力量？

<br/>        记名弟子兴奋不已，后面坐着的妖王们却是两腿打颤，要知道，燕山能够拥有如今这般实力，靠的就是血衣剑卫和炼丹房这两样大杀器，但是现在，所有的一切，对于记名弟子全面解封，意思就是说，会出现三千六百个燕山这样的庞然大物！

<br/>    
<br/>        那是一片足以吞噬一切的红色潮水！

<br/>        他们已经亲眼见证了燕山的崛起，对于燕山的模式深信不疑，而现在，又有三千六百座燕山要兴起了吗？

<br/>        拜月此刻可是看到了商机，张鸣这小子可是第一个突破到入道境的，而且，现在这些记名弟子可是相当于一棵棵燕山幼苗，可是有机会成长为燕山这样的庞然大物的，这一百万灵石，绝对不亏！
