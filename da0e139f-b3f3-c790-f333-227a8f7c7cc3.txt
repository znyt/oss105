　　轰…轰…轰…
<br/>　　一条条武道长河，在北隐宗大长老的掌控下，狂猛朝萧逸袭去。
<br/>　　萧逸怡然不惧，一拳轰出。
<br/>　　拳头，手臂上，火焰爆发，环绕其中。
<br/>　　惊人的力量，一拳所过，武道长河轰然破碎。
<br/>　　萧逸双手齐出，每一拳所出，必是火光环绕，爆鸣剧烈。
<br/>　　而每一声爆鸣之下，又必是一条武道长河被轰散。
<br/>　　九千六百道武道长河连绵不断绞杀而来，其结果，却仅仅是不断被轰散，丝毫未能奈何萧逸。
<br/>　　“怎么可能。”远处，北隐宗大长老面露骇色。
<br/>　　“那双手臂，是兽化？”
<br/>　　“世人皆传你萧逸小贼乃是控火兽废武魂，原来竟是别的妖兽，还强大如斯。”
<br/>　　“兽化？”远处，萧逸闻言，先是一愣，随后冷笑一声。
<br/>　　不错，他这时的双手，看起来像极了兽化。
<br/>　　所谓兽化，便是兽武魂拥有者，调动武魂力量，让得武魂虚影实体化作用于自身。
<br/>　　这对于兽武魂武者来说，是再寻常不过的事。
<br/>　　以前萧逸也对战过不少兽武魂武者，往往兽化，凝聚武魂实体时，便是这个兽武魂武者实力尽出之时。
<br/>　　不过，萧逸现今可不是兽化。
<br/>　　兽化，靠的是武魂力量。
<br/>　　现今这双兽手，可是真正的妖兽尸骸，不过是通过八龙焚火炉的力量，幻化覆盖到他手上罢了。
<br/>　　嘭…
<br/>　　忽然，空气中一声爆响。
<br/>　　萧逸左脚猛地一瞪。
<br/>　　恐怖的爆发力下，身影瞬间横跨飞跃。
<br/>　　一个呼吸间，萧逸便横穿了层层武道长河。
<br/>　　哗…哗…哗…
<br/>　　一条条武道长河，霎时溃散。
<br/>　　一股股水流，从长河中奔溃、溅落，再到消散无踪。
<br/>　　“怎么可能，好快的速度。”北隐宗大长老瞪大了眼睛。
<br/>　　这是他在短短时间内就连续说出的好几次‘怎么可能’了。
<br/>　　在他眼中，刚才的萧逸，不过是脚步一瞪，便忽然化作了一道火焰流光。
<br/>　　火焰流光一冲而过，瞬间将他的一道道武道长河冲散。
<br/>　　“怎么，堂堂北隐宗大长老，就这点儿本事？”
<br/>　　“就只会说怎么可能？”
<br/>　　萧逸漂浮在高空中，冷笑一声。
<br/>　　狰狞的笑容中，带着冰冷的漠视。
<br/>　　北隐宗大长老先是一愣，随后暴怒，“臭小子，在老夫面前，还轮不到你猖狂。”
<br/>　　愤怒的宣言，夹杂着天地之威，仿佛神祗之言。
<br/>　　滔天气势，瞬间从北隐宗大长老身上爆发。
<br/>　　气势之息，连接天地气息，宛若天地之势。
<br/>　　方圆百里，霎时风云变幻。
<br/>　　天空中，滚滚怒雷渐成；地面上，川河倒流，山峰崩塌。
<br/>　　远处，萧逸看着这一股股恐怖气势，双眼一眯。
<br/>　　他知道，现今的北隐宗大长老，已然彻底暴怒。
<br/>　　而在这暴怒之下，便是北隐宗大长老的实力爆发。
<br/>　　一个暴怒的武道大能，实力尽出，其可怕程度，不难想象。
<br/>　　武道大能，已经是一方大能，有参天手段。
<br/>　　历史上，留有记载的武道大能间的战斗，并不多。
<br/>　　但每一份记载上的战斗，必是天崩地裂，宛若末日。
<br/>　　现在，一个武道大能，已然暴走了。
<br/>　　但，与之交手的，却不是相同的一位武道大能，而仅仅是一个二十来岁的年轻人。
<br/>　　“发飙了吗？”萧逸冷笑一声。
<br/>　　他自然知晓愤怒下的武道大能有多可怕。
<br/>　　但，他丝毫不惧。
<br/>　　双手上，黝黑鳞甲内所蕴含的力量，给了他足够的自信。
<br/>　　他不知道该怎么形容这种感觉。
<br/>　　只是直觉告诉他，现在的他，双手仿佛拥有无穷无尽的力量。
<br/>　　那有力的手臂，仿佛能捏碎天地。
<br/>　　刚才那些朝他袭来的武道力量，他之所以毫不犹豫地出手捏之。
<br/>　　便是因为他直接那些所谓的天地规则之力，在他双手之下，那一瞬间，显得脆弱不堪。
<br/>　　“臭小子。”北隐宗大长老此刻的气势，已然达到顶峰。
<br/>　　武道大能的一身实力，展现得淋漓尽致。
<br/>　　山川河岳，尽在他一身气势下，颤抖不停。
<br/>　　“老夫不仅要将你埋尸金光禁地，还要你死无全尸。”
<br/>　　冰冷而愤怒的话语，响彻天际。
<br/>　　萧逸怡然不惧，“有本事便来。”
<br/>　　嘭…嘭…嘭…
<br/>　　一道道紫色火焰凭空而现，随后幻化火海。
<br/>　　火海之中，一条条紫炎游龙，狂猛而出。
<br/>　　北隐宗大长老的掌风，已然打出。
<br/>　　紫炎游龙与掌风瞬间碰撞。
<br/>　　轰…轰…轰…
<br/>　　武道大能的全力一击，确实厉害，掌风所过，一条条紫炎游龙被轻松轰散。
<br/>　　但不过数秒。
<br/>　　上千条紫炎游龙还未完全被轰散之时，掌风威力已然停滞。
<br/>　　恐怖的高温火焰，瞬间将掌风焚烧殆尽。
<br/>　　“嗯？紫晶灵炎？”北隐宗大长老脸色一变。
<br/>　　“你那兽化的武魂，拥有操控紫炎之力？”
<br/>　　“不对…”北隐宗大长老，似是想起了什么。
<br/>　　“大半年前，四方城那边似乎出了个名动一时的控火天骄，名唤易霄。”
<br/>　　“一手紫炎，败尽四方域各大天骄，连十八府天骄，包括十六尊使之一都一一落败。”
<br/>　　“老夫没记错的话，你萧逸，也是在四方域成名的吧。”
<br/>　　“你成名之时，便恰好是易霄销声匿迹之时。”
<br/>　　“原来如此。”北隐宗大长老双眼一眯，随后愈发冷笑。
<br/>　　“萧逸，易霄，看来老夫今日要一连让两位绝世妖孽陨落。”
<br/>　　“痛快，痛快。”
<br/>　　话音落下，北隐宗大长老的气势，再次爆发。
<br/>　　一道道元力，自成游龙，倾泻而出。
<br/>　　元力，异常浑厚，显然便是北隐元劲。
<br/>　　北隐元劲，在武道大能的一身精纯修为下，愈发惊人。
<br/>　　再加上一道道天地规则之力加身。
<br/>　　这一条条元力游龙，霎时张牙舞爪，宛若绞碎天地。
<br/>　　嘭…嘭…嘭…
<br/>　　萧逸双手齐出，一个个紫炎火海，充斥整个高空。
<br/>　　火海之内，数以千计的紫炎游龙，带着吞天之势，狂猛而出。
<br/>　　紫炎游龙与元力游龙间的交锋，霎时而起。
<br/>　　一是来自于世间强悍火焰，紫晶灵炎所化。
<br/>　　且由那双不知名的兽手的恐怖控火之力而增幅。
<br/>　　一是武道大能的精纯元力，加北隐元劲所化。
<br/>　　且有无数武道力量增幅。
<br/>　　二者的交锋，游龙的缠绕，让得金光禁地的高空霎时崩塌。
<br/>　　激扬的元力，恐怖的火焰，交汇处一副宛若末日的景象。
<br/>　　一声声轰鸣，一道道惊天爆响，响彻不停。
<br/>　　整整数分钟后，爆响声忽然结束。
<br/>　　北隐宗大长老暴怒的脸色下，忽然化作一抹痛苦之色。
<br/>　　他那苍老而锐利的眼眸，不可置信地看了眼自己的胸膛。
<br/>　　胸膛处，不知何时起，已被一只狰狞的黑色兽手所洞穿。
<br/>　　兽手上，散着寒芒的锋锐爪子，骇人至极。
