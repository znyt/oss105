　　第541章
<br/>　　二人一番忙碌，终于将尸体埋葬，事实上，也是李莫愁忙碌，李婉儿并没有出多少力。
<br/>　　天快黑了，西方一轮红日失去了正午的炙热，给人一种冷冰冰的感觉。李婉儿立在新堆的坟墓前，双手合十缓缓祷告。
<br/>　　做一件事未必有一件事的好处，但至少能给李婉儿一些心理安慰的。如果今天的乱世，是李家造成的。那么自己多做一件事，就能多积一些功德，赎一分李家造成的罪孽。
<br/>　　李莫愁打量着李婉儿，她当然看不惯李婉儿的性格。可这一路走来，却也佩服李婉儿身上展现的韧劲儿。
<br/>　　“你有没有想过，如果你真的想做一些事，其实有更好的办法。无论怎样做，都比你现在做的有意义。”李莫愁还是忍不住问了出来。
<br/>　　李婉儿没有做声，默默的看着面前的新坟，心里呐呐了一句：其他我又能做什么呢。
<br/>　　这时候，一老者拄着木杖从远处过来，经过李婉儿身边时，似乎对她的所作所为的有些奇怪，停住了脚步打量着二人，轻轻咳嗽着。
<br/>　　“老伯。”李婉儿行了一礼，问道：“天快黑了，老伯一个人赶路，不知要去哪里？”
<br/>　　这老人便是孟玄青，年纪大了，腿脚自然慢些，他长舒口气，道：“我想去琅琊城看看，小妮子，你们两个要去哪儿呢？”
<br/>　　“我们……”李婉儿苦笑一声：“我们也不知要去哪儿，不如我们同路，一起去琅琊看看。”
<br/>　　李莫愁倒是没有李婉儿这样轻信于人，她盯着老人，其实也没有看出什么。但潜意识里觉得对方有些危险。
<br/>　　孟玄青呵呵笑笑，没有答应，也没有拒绝，只是拄起木杖，继续向东走。
<br/>　　李婉儿和李莫愁缀在他身后，也没有跟得太紧，双方大概相距百步距离。
<br/>　　李莫愁冲李婉儿低声道：“这老头古古怪怪，怕来历不简单。普通老人，怎么敢独自上路，我们还是离他远些。”
<br/>　　“一个老人，能有什么危险。”李婉儿叹口气：“我们跟着他，如果他突然倒在地上，我们正好埋了他。”
<br/>　　“呃……”李莫愁看看肩上两柄铁锹，埋人这种事也有瘾么。
<br/>　　……
<br/>　　红胡子可以说是东海海域上，最强的几股海盗之一。程大雷肯定还对他有些印象，他还有一个名字，叫做耿星河。当然，这是本名还是化名，就无人知道了。出来打家劫舍，赶着掉脑袋的活，大部分人是不会用爹娘给的名字。
<br/>　　当初程大雷在扬州借粮的时候，红胡子还想打程大雷的主意，后来自然是损兵折将惨重，不过这些年过去了，他已经恢复了元气，占据骷髅岛，手下有两三千弟兄，什么倭人、爪哇人。每日出海抢劫，日子过得不亦乐乎。
<br/>　　但最近，红胡子的日子有些惨。
<br/>　　落凤城来了一伙人，也不知道怎么搞得，让一群老弱病残像吃了药一样猛。每日驾着战船，在海上晃荡，遇到海盗就抢。
<br/>　　当然，偶尔也抢商船。
<br/>　　比海盗还海盗，这帮人难道就没有底线么。
<br/>　　今日，红胡子的海盗船与落凤城的剿匪部队在大海上不期而遇。
<br/>　　戚继光立在船头，手撑着宝剑，口中呼喝出声。
<br/>　　“一颗海盗的脑袋十两银子，怯战后退，斩立决。”
<br/>　　双方的大船互相冲锋，以求近身一战。
<br/>　　李行哉从船舱中走出来，用大氅裹紧身子：“戚将军，此战有多大把握，据说这红胡子实力可是不弱。”
<br/>　　“王上放心，一群乌合之众而已。何况，咱们这次遇到的只是他们的小部队，只有两艘船，一百余人，拿下他们不会有多大问题。”
<br/>　　李行哉站在甲板向前方望，虽然他经历了很多事，但海战的确是第一次。此刻大海上风平浪静，天高云远，倒是个杀人的好天气。
<br/>　　“海上风大，王上不如回舱内休息，由我料理此战，不用费吹灰之力。”
<br/>　　“放心，我也不是暖房中的花草，受不得风吹浪大。我且看看，这一战如何打得他们落花流水。”
<br/>　　李行哉来到落凤城这段时间，今日算是站稳了脚跟。这一战可以算是扬名一战，打好了，继而就可以攻击骷髅岛。红胡子在大海上盘踞这么久，岛上该有多少宝物，都可以用作招兵买马的本钱。
<br/>　　如果打不好，那就只有回去继续窝着，辛苦攒下来的家底说不得也会折进去。
<br/>　　双方的战船对撞，在两舷相接的时候，双方互相用长矛攻击，羽箭飞来飞去。在远程武器用上之后，便是白刃战。
<br/>　　李行哉一抖身上大氅，拎起铁枪，口中大吼一声，跳上了对面的战船。
<br/>　　“弟兄们，与我冲。”
<br/>　　陷入海盗群中，李行哉一招画龙点睛，招招都去扎敌人的喉咙。
<br/>　　“打他下盘，他的下盘不稳。”海盗大喝。
<br/>　　李行哉有些微瘸，步法上有些问题。当然，他也没如何练过步法，也是经过多番磨难，本事才一点点强起来。
<br/>　　此刻他站在那里如一根柱子一般，铁枪一点一扎，身周已倒下三五具尸体。
<br/>　　漕帮的李佳明等人，本不服李行哉，见到这一幕，心中微微有些吃惊。
<br/>　　“弟兄们，与我杀啊！”李佳明舞着大刀，一刀将面前海盗脑袋砍成两半。
<br/>　　戚继光却是吃了一惊，怎么一会儿没看稳，逍遥王就亲自上阵了。
<br/>　　“王上，小心！”
<br/>　　戚继光也提着枪冲了过去。
<br/>　　“来，与我并肩杀敌。”
<br/>　　李行哉将脸上的血水抹去，一枪穿了海盗的喉咙。
<br/>　　戚继光微微呆了呆，大武开国皇帝，用一柄枪扫平南北。今日的李行哉，已有几分祖先威风。
<br/>　　在他身上，已露出麟角之象。只等着一遇风雨，便化龙而起。
<br/>　　一场鏖战，大海上漂着几具尸体。李行哉兵力占优，又有戚继光这样的大将，胜利自然没有半点悬念。
<br/>　　最后抢了他们两艘大船，杀人无数，只不过海盗的魁首红胡子跳海逃走了。
<br/>　　鏖战过后，李行哉立于船头，目光遥遥望着远方。
<br/>　　下一步，就是骷髅岛了。
