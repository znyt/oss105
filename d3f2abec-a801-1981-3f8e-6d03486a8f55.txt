　　对于现在的什造来说，让她参加这种级别的战斗，风险也实在太高，更不要说，什造身上的伤势，到现在还没有恢复完全。
<br/>　　“唉，那这样的话，我们就没法谈咯。”
<br/>　　萝玛从办公桌前站起身，随后，身体开始变化，迅速扭曲成一只和蜘蛛有八分相似的狰狞怪物。

<br/>　　赫者化形态，混乱之母。

<br/>　　还算空旷的办公室内，灯光不是很亮，刘哔手持着双刀，迎上了前方恐怖的怪物。
<br/>　　两人的交手速度都很快，一时间，房间里充斥着刀光人影，不时洒下一些血液，将地面腐蚀出一个个小洞。
<br/>　　令人头皮发麻的“滋滋”声，从地面上传来，这些颜色有些发暗的血液，竟带走如此强大的腐蚀性。
<br/>　　萝玛仰着一个肉球般的脑袋，半边脸都被血盆大口所遮掩，口中，是一圈令人发寒的锯齿，如同放大了许多倍的巨型蜘蛛一般，正挥舞着双手，拼命抵挡着刘哔的进攻。
<br/>　　让人感觉怪异的是，在赫子怪物的头顶上居然有着一个小女孩的上半身，此时正狂笑的看着众人，脸上的表情混乱而疯狂。
<br/>　　此时的帆系萝玛心中也是暗暗叫苦，本来她以为凭借着自己的实力，虽然无法干掉刘哔，但拖一段时间还是很轻松的事情。
<br/>　　可没想到，这家伙的速度实在太快了，而且力量更是非凡，她甚至连有效的反击都组织不起来。
<br/>　　这个家伙，难道是怪物吗？
<br/>　　帆系萝玛抱着这样的念头，咬牙苦苦支撑着，硬生生将刘哔拖在了这里。
<br/>　　“唰！”
<br/>　　就在两人战斗正酣的时候，办公室的窗户突然被击碎，两道人影迅速从窗外跃入，丝毫没有犹豫的向刘哔发动了袭击。
<br/>　　一根粗长的赫子，以及一柄长剑，擦着刘哔脸颊而过，带起了一丝血花。
<br/>　　这是他在今天的战斗中第一次受伤，这两个突然出手的敌人，挑选的实在是太好了，刚刚和萝玛对拼一记，他根本无法完全躲开。
<br/>　　如果换成是别人，估计脑袋都要被人给捅穿了。
<br/>　　上来就直接攻击较难命中的头部，这些人看来是已经将他给研究通透了。
<br/>　　“哟，萝玛，怎么搞得这么狼狈。”门外，传来诗的声音：“话说，我没有来晚吧。”
<br/>　　虽然知道萝玛的状况十分紧急，但是为了不引起刘哔的戒备，他并没有选择第一时间动手，而是等人员全部就位之后，才匆忙赶了过来。
<br/>　　开玩笑，说什么要将小丑组织交给他管理，他才不稀罕这些东西。
<br/>　　和萝玛比起来，这个世界的其他东西，都算不上什么了。
<br/>　　嘴角勾起一抹笑容，刘哔转头，看到了鱼贯而入的一众人群。
<br/>　　这些人，除了诗之外，全都是清一色的红色独眼，穿着一身乳白色的制服，胸前别着一个世界树的金色标志。
<br/>　　不过，这些白衣服的人，他都识得，就在前不久之前，他还在这些人的总部别墅。
<br/>　　白日庭，CCG的特种作战搜查官们，带领这次行动的，正是教皇。

<br/>　　对他做了大量的研究，和修家族对他的了解十分深刻，在和修家的人眼中看来，刘哔就是一个为了变强而不断共喰的特殊喰种。
<br/>　　枭讨伐战，月山家驱逐战，以及这段时间零零散散死在他手中的喰种，这所有的一切加起来，已经足有几百条喰种的性命葬送在他手中了。
<br/>　　这还不算上那些死去的搜查官，要知道从他参与第一次行动到现在，方才过去了多长时间。
<br/>　　而这些死去的喰种，无一不是自己区域内的最强者，但只要是被他盯上，就算是SS级喰种都无法逃脱死亡的命运。
<br/>　　和修家断定，之所以如此疯狂的共喰，必然会存在一些特殊的原因。
<br/>　　或许，是来自国外，或者，就是受了很严重的伤势，需要共喰才能进行恢复。
<br/>　　不管是哪一种，和修家都不会继续这样放任下去了，刘哔的存在，已经对他们造成了很大的威胁。

<br/>　　“为了对付我，你们还真是煞费苦心啊。”刘哔舔了舔嘴唇，带着侵略性的目光环视了一些四周，最后落在了萝玛身上。

<br/>　　察觉到刘哔的目光，原本趴在头顶上的萝玛直起腰来，脸上带着充盈的笑意。
