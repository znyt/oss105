<br/>    无冕者议会的祭祀还会停留在天庭，四大仙国，特别是天庭，也是在此刻招揽四方的游方老道，希望能够有窥测天机的高人出手，治一治这个无冕者议会的祭祀。

<br/>        一时之间，无冕者议会风头无两。

<br/>        没毛病！

<br/>        毕竟这玩意儿不提升任何战斗力。

<br/>        “哎呀，这么惨的啊，不知道天庭有没有什么其他的办法。”

<br/>        陈明思索了片刻，自己好像可以装作一个游方僧人，过去度化这些亡魂，而后混进佛门之中，佛门之中，可是也有着能够揣测天机的和尚。

<br/>        这种和尚，便叫做修心僧。

<br/>        不过，这个冒充，也太伤了，要牺牲自己的美貌，剃光头！
<br/>    
<br/>        不过好像对于修行者来说，这也没什么，毕竟长个头发嘛，对于陈明来说也就是一瞬间的事，
<br/>    
<br/>        有着《太素灵棺》，血肉都能重生，别说头发了。

<br/>        陈明回到自己的房间之中，拿出一把菜刀将自己的头发剃了个干净，伸手一摸，滑不溜秋，照了照镜子，锃光瓦亮的一颗大光头！

<br/>        然后陈明换上一身素白衣裳，俨然一个俊俏的年轻和尚。

<br/>        是时候了，去看看吧，就算做好事了。

<br/>        这仙路，自己要怎么过去？

<br/>        飞过去不就暴露了？

<br/>        怎么过去呢，这是一个问题。
