<br/>　　不过，惊讶归惊讶，叶尘的表现，还不足以让他们感到畏惧。
<br/>　　且不说那两位灰袍老者，乃是化神后期的实力，便是韩月公子，虽然只有半步化神境，也具备秒杀之前那三个人的实力。

<br/>　　韩月公子说到这里，微微一顿，冲着一旁其中一名老者，冷声道:
<br/>　　“宫老，杀了他！！”
<br/>　　“是！”
<br/>　　那名老者缓步走上前来，一脸不屑的盯着叶尘看了片刻，
<br/>　　“小子！以你只有元婴境四重的修为，可以做到这种程度，的确算得上是一个绝世天才！放眼整个擎苍宗，也只有当年的绝无敌公子，有此等天赋！”
<br/>　　“倘若再给你几十年的时间，恐怕我们几个都不是你对手！”
<br/>　　“可惜呀，做人还是低调点的好，你太嚣张了，注定走不到那一步！”
<br/>　　那灰袍老者说完这话的同时，身上的气息也开始节节攀升，
<br/>　　轰隆！
<br/>　　一股令人心悸的气势，冲天而起！
<br/>　　那灰袍老者的身上，竟然升起一道高达近千米的金色虚影。
<br/>　　不愧是化神后期的强者，单单是一个外道神像的神通，竟然就已经恐怖如斯！
<br/>　　甚至此时，连谷外的冥沧荨等人，也全都看到了灰袍老者祭出的外道神像，一个个全都满脸骇然之色。
<br/>　　“不愧是擎苍宗的强者啊！此人好像只是韩月公子的随从，放眼我们整个南斗星域，恐怕都无一人是其对手！”
<br/>　　“幸好之前，我们没有和那位韩月公子发生冲突，否则后果真是不堪设想！”
<br/>　　“那小子这次，肯定死无全尸了吧！”
<br/>　　……
<br/>　　三大家族的人纷纷议论着，而冥沧荨早已经娇躯颤抖不止，俏脸惨白无比。
<br/>　　即便是隔着老远，冥沧荨也能感受到，
<br/>　　对方太强了！
<br/>　　甚至比冥家大长老还有强大的多！
<br/>　　而且她刚刚听三大家族其他人话里的意思，似乎这个等级的强者，还不止一位！
<br/>　　面对如此强大的存在，叶尘，还能活命吗？
<br/>　　就在冥沧荨心惊胆颤之时，九天之上，忽然响起了叶尘那淡漠的声音，犹如雷音滚滚而来，
<br/>　　“你们若是一起出手，或许还能与我一战，可惜啊！你们根本不知道，自己面对的是什么样的存在！”
<br/>　　随着这道声音响起，
<br/>　　下一刻，
<br/>　　在众人无比震撼的目光中，天空之上陡然出现了八道硕大的拳头，同时朝之前那道高达千米的外道神像轰击而去！
<br/>　　“八极镇魔神拳！！”
<br/>　　叶尘直接暗暗施展出逆生百转，修为暴涨了数十倍之后，又果断使出了这门帝级拳法。
<br/>　　“什么！！”
<br/>　　等到后方的韩月公子，和另外一名灰袍老者，两人反应过来的时候，显然已经为时晚矣。
<br/>　　轰！轰！轰！轰！轰！轰！轰！轰！
<br/>　　那八道犹如洪荒凶兽一般的拳头，狠狠砸落那位宫老的外道神像之上。
<br/>　　彭！
<br/>　　没有任何悬念，之前三大家族感到心悸的那尊外道神像，竟然犹如纸糊的一般，瞬间破碎开来。
<br/>　　继而，
<br/>　　“啊！！”
<br/>　　伴随着一道凄惨的吼叫声响起，那位宫老，堂堂化神境后期的强者，竟然被生生轰成了一堆肉渣，唯独一道勉强逃脱的残魂，飘荡在九天之上，发出一道道凄厉的惨叫，继而随风慢慢湮灭。
<br/>　　“什么！”
<br/>　　“这……”
<br/>　　所有人包括韩月公子在内，看到这一幕之后，彻彻底底被震撼到了。
<br/>　　那可是化神境后期的强者，而且还是来自大名鼎鼎的擎苍宗，一人足矣横扫整个南斗星域的存在啊！
<br/>　　竟然就这样，被他们看不起的年轻小子，一招给秒了？
<br/>　　难道他之前故意扮猪吃虎，实则却是一位返虚真君不成？
<br/>　　可是他真元波动，明明就是元婴之力啊？
<br/>　　“咕噜~”
<br/>　　韩月公子想到这里，不由得狠狠咽了一口口水，早已经满脸惊恐之色。
<br/>　　他打死也没有想到，宫老居然就这么被眼前这个，名不见经传的小子，给活活打爆了！
<br/>　　要知道，即便是他们擎苍宗这一代，被誉为千年一遇的绝世天才，在没有迈入化神境之前，也不具备如此恐怖的实力啊！

<br/>　　韩月公子下意识的倒退了数步，满脸骇然的望着叶尘，哆里哆嗦的开口问道。
<br/>　　他是真的怕了！
<br/>　　虽然他怎么也想不明白，眼前这个看上去，比他还要年轻几岁的小子，为何会具备如此恐怖的战斗力。

<br/>　　他本以为报出自己的身份，对方多少会有所忌惮，可是怎么也没有想到，对方竟然连擎苍宗都不放在眼里，

<br/>　　韩月公子忽然冲着另外一名灰袍老者一声暴喝，而自己则转身就逃。

<br/>　　如今叶尘的修为，比之前在云丹城，又提升了一大截，在施展出逆生百转的秘术之后，即便是返虚真君，也有一战之力，更可况是他？
