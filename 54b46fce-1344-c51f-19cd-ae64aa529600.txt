　　其他人也几乎和江飞燕两人，全都是相同的看法。
<br/>　　叶尘以一己之力，逼的美国俯首，此事虽然在网络上大为流传，可毕竟太过于玄幻了。
<br/>　　除了少数高层，具体知道什么情况之外，普通人根本就是一知半解，而且很多人并不知道，华夏仙人便是叶尘！
<br/>　　至于华夏军方授予叶尘元帅军衔，也是秘密授勋，别说普通人不知道，便是军中高层，知道这件事的也不多。
<br/>　　所以，这位突然出现的上校，虽然对叶尘口称叶帅，但很多人并不认为，就是元帅的意思。
<br/>　　或许，只是一个特别的称呼而已。
<br/>　　毕竟，就算华夏国真的有元帅，也绝对不可能会是，一个二十岁不到的少年啊！
<br/>　　然而，
<br/>　　这少年接下一开口，却让众人大跌眼镜。
<br/>　　只见他一脸平淡，道：
<br/>　　“小张，你怎么来了？在警局大喊大叫，成何体统？”
<br/>　　原来，眼前的这位上校，赫然正是之前，受易山河指派，为他安排在天海大学就读的那位！
<br/>　　叶尘只记得他姓张，具体叫什么，他之前也没有问。
<br/>　　众人见这少年，竟然当众训斥一名上校，顿时全都看懵了。
<br/>　　然而，让众人更加懵逼的是，这位上校竟然一点不悦的表情都没有，当即恭恭敬敬道：
<br/>　　“是，叶帅，我知道错了！不过，到底是哪个不长眼的抓了您，此人绝对不能轻饶！”
<br/>　　张上校这话一出口，一旁的江飞燕，顿时脸色微微一白，旁边的那名青年警察，更是出了一身冷汗！
<br/>　　他们这些警察，最怕的就是和军队打交道，抓又抓不得，打又打不过，而且军队有一个最大的毛病，就是极其护短！
<br/>　　所以，他们平时碰上军队的人，都要退避三色，轻易不敢去招惹，否则分分钟给你拉来一支真枪实弹的队伍。
<br/>　　看到张上校动怒，不但江飞燕两人，心中忐忑不安，就连一旁的张局也眉头紧皱，露出一抹凝重之色。
<br/>　　不料，
<br/>　　叶尘听到这话，却随意摆了摆手，道：
<br/>　　“算了！此事因我而起，他们也只是按照章程办事，你不得再招惹事端！”
<br/>　　张上校当即身体一正，
<br/>　　“是！”
<br/>　　而听到叶尘这么说，张局和江飞燕等人，也全都纷纷松了一口长气。
<br/>　　就在这时，萧若曦和白玲两人，也被警察带了过来。
<br/>　　萧若曦早就知道，以叶尘的身份，他们绝对不会有事，所以倒还不觉得有什么。
<br/>　　白玲看到眼前的架势，已经有些懵了，完全不知道发生了什么。
<br/>　　“既然人都到齐了，那就走吧！”
<br/>　　叶尘也不废话，直接一把拉住萧若曦的手腕，朝外面走了过去，张上校和张局，全都快步跟上，其他众人看到张局都亲自恭送，于是也纷纷跟着后面。
<br/>　　一旁的白玲，看到这一幕，已经彻底傻了，
<br/>　　“就这么把我们放了？而且看着阵势，怎么反倒像是某位大领导视察似得？这到底什么情况？”
<br/>　　“难道是若曦动用了萧家的力量？”
<br/>　　“可即便是那位广南首富亲至，也搞不出这么大的阵势吧？”
<br/>　　白玲随波逐流跟着后面，心中百思不得其解。
<br/>　　……
<br/>　　出了警察局之后，三人上了一辆军用吉普车。
<br/>　　“叶帅，我先送您回学校吧！”
<br/>　　张上校坐在叶尘旁边，屁股只坐半边，腰挺的笔直，恭恭敬敬的请示道。
<br/>　　坐在后面的白玲，此刻也终于看出了一丝端倪。
<br/>　　他们之所会这么快被释放，好像全因为这位张上校，而这位张上校，似乎又是冲着叶尘的面子。
<br/>　　而更让白玲感到骇然的是，
<br/>　　这位张上校，在叶尘的面前，简直恭敬的不像话，就像是小学生面对老师一般！
<br/>　　‘这家伙到底有着什么样的背景？何以可以让一位堂堂上校，对他如此恭敬？’
<br/>　　猛然间，
<br/>　　白玲又突然想起，萧若曦曾经对朱博文说的那番话，
<br/>　　“不是他配不上我，而是我配不上他！”
<br/>　　“你对他，根本一无所知！”
<br/>　　想到这里，白玲忍不住偷偷朝前面的叶尘瞄了一眼，同时又望了一眼叶尘旁边，那位满脸崇敬之色的张上校。
<br/>　　也不知道是不是心理作用，白玲忽然觉得眼前这个，比她还要小上两三岁的少年，似乎有着一种难以形容的独特气质，使她心颤不已，
<br/>　　‘莫非他是某位超级大家族的公子哥？对！一定是在这样！’
<br/>　　一念及此，白玲顿时又懊悔不已，
<br/>　　‘早知道他有如此强大的能量，即便他是若曦的男友，我说什么也要争取一下，可惜现在，已经彻底晚了！’
<br/>　　……
<br/>　　另外一边，江北区警察局，
<br/>　　“张局，那小……那位叶帅，到底是什么来头啊？”
<br/>　　将叶尘等人送走之后，江飞燕实在按耐不住自己的好奇心，向一旁的张局开口询问起来。
<br/>　　张局望着那辆渐行渐远的吉普车，瞥了一眼旁边的江飞燕，似笑非笑道：
<br/>　　“叶帅，自然就是叶帅喽！还能是什么来头？”
<br/>　　江飞燕微微一愣，旋即便明白了张局话中的意思，顿时两眼一瞪，
<br/>　　“张局的意思是，他真的是我们华夏国的……”
<br/>　　“可是，我国如今最高军衔不是上将吗？连大将都已经几十年没有了，怎么会突然冒出一个元帅？而且还如此年轻！”
<br/>　　张局深吸了一口气，方才缓缓道：
<br/>　　“是呀！华夏国已经几十年没有打仗了，早就没有了元帅。”
<br/>　　“如果换做是别人，我肯定打死也不相信，可如果是他，我却觉得没有好奇怪的！”
<br/>　　江飞燕顿时更加懵了，
<br/>　　“张局，麻烦您能不能说明白一些，我怎么越听越听不懂了？”
<br/>　　张局微微一笑，忽然话锋一转，
<br/>　　“小江啊，你可听说过华夏仙人？”
<br/>　　一提及这个，江飞燕顿时来了精神，
<br/>　　“当然！据说我华夏出了一位仙人，一人横扫了米国数万军队，捣灭了米国在太平洋上的海军基地，以肉身硬抗核弹，逼迫米国认输投降……”
<br/>　　说到这里，江飞燕顿时美眸一亮，似乎想到了什么，
<br/>　　“张局，您不要告诉我，他就是那位……”
<br/>　　张局笑着点了点头，
<br/>　　“不错！他就是那位华夏仙人！而且，他还有另外一个名号，云州叶先生！”
<br/>　　“什么！”
<br/>　　江飞燕顿时目瞪口呆。
