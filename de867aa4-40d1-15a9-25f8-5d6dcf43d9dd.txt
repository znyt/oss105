　　第548章
<br/>　　“哎呀这个么。”程大雷抿了口茶：“倒是从未听说过。”
<br/>　　“那魔头一直在凉州肆虐，这次不知为何来了东海。”夫子叹了口气：“如今这件事已传得沸沸扬扬，更有传言说，他千里来东海，便是冲着我书院来的……呃，这位朋友是有眼疾么？”
<br/>　　程大雷轻咳一声，制住不断眨眼的刘发财，道：“夫子慧眼如炬，一眼就看穿了。不用理他，咱们聊咱们的，那程……魔头和书院也该没什么梁子，他为何要冲书院来？”
<br/>　　“抢我书院的弟子啊。”夫子理所当然道：“天下谁不知万卷山人杰地灵，他定是相中了我书院的弟子，千里迢迢过来，就是抢走我书院的学生。况且他一路东来，杀人放火无算。我书院读圣贤书，行仁义事，便要替天下人除了这魔头。”
<br/>　　“哎呀呀，夫子言之有理。想这样的魔头早就该杀，早晚要杀，不杀不以除民愤，不杀不足以匡正道。”刘发财咬牙切齿，怒发冲冠。
<br/>　　“这位先生……像是对程魔头有很深的仇恨。”夫子皱了皱眉头，道：“所以才有这次伏魔大会，我书院出头，邀请东海一地的高手，商定一个消灭魔头的计划。欧公子既然有如此本事，明日伏魔大会上可不要藏私，一定要在诛杀这魔头时出一份力。”
<br/>　　“嗯，这个一定一定。”
<br/>　　程大雷支应两句，随之起身告辞。有小童过来，引着三人来到后院厢房。
<br/>　　书院派人送来饭菜，待客之礼还是很周到的。食不厌精脍不厌细，饭菜也做得很精致。程大雷和刘发财一直挤眼，二人有很多话要聊，只不过当着楚云生不方便说出口而已。
<br/>　　楚云生显得有些心不在焉，今天一路闯关斩将，也见到了传说中的夫子，经历宛若一场梦而已。如今这一切过去，他变得忐忑起来。
<br/>　　“书院会不会报复我？”楚云生问：“将我挤在没人的地方揍一顿？”
<br/>　　“怎么可能嘛，书院也不是心眼小的地方。”程大雷随口道。
<br/>　　“怎么不可能。”楚云生想起白日的事：“他们的心眼还不够小么？”
<br/>　　咚咚咚。
<br/>　　外面想起敲门声，楚云生一激灵，筷子差点掉在地上。
<br/>　　“他们这么快就来了？”
<br/>　　“嗳，枪打出头鸟，你想出头就要扛一扛风雨。”程大雷冲着门口喊了一声：“谁？”
<br/>　　“楚公子在么，东海王门下朱力特来拜见。”门外有人答道。
<br/>　　门内三人对视一眼，帝国封十八路诸侯王，单单徐州一地就有三位，分别是徐州王、东海王、逍遥王。
<br/>　　琅琊城算是东海王的领地，只是不知他要见楚云生为何。
<br/>　　楚云生在得到程大雷的眼神鼓励后，起身将门打开，见外面站着一宽袍大袖的锦衣中年人。
<br/>　　“闻听楚公子到了此地，在下准备了晚宴，特意招待楚公子。”
<br/>　　楚云生一头雾水，目光看向程大雷和刘发财。
<br/>　　“人家让你去就去么，见见面聊聊也是好的。”程大雷有心支开楚云生。
<br/>　　楚云生这才和朱力一同起身，正走着只见迎面有来了三四人。
<br/>　　“是楚公子么，在下是徐州王的门人，特意送来一份薄礼，请楚公子笑纳。”
<br/>　　程大雷远远望着，见楚云生没走几步就被堵住了。都是各路诸侯的代表，有送华服的，有送佩剑的，有送金银玉石的，还有送美妾的，水灵灵的眼神望着楚云生。
<br/>　　程大雷和刘发财啧啧称奇，难掩眼神中的羡慕。不提明日即将到来的伏魔大会，各路高手要到琅琊城。便是平日，各路诸侯也在此地安插着眼线，专门负责拉拢人才。
<br/>　　今日楚云生小露了一手，已进入了各路诸侯的眼界，如这般数算天才，自然是他们拉拢的对象。不惜血本，就算在书院大打出手也在所不辞。
<br/>　　“瞧瞧，咱们是和他一起来的，怎么没人来拉拢我们。”刘发财心有不甘。
<br/>　　“大概觉得咱们是他的随从吧。”程大雷道：“拉拢到他，就等于拉拢到我们。”
<br/>　　“啧。”刘发财将门关上：“看来他今晚是不会回来啦。”
<br/>　　的确，各路诸侯齐献殷勤，美酒美女美食，他估计想回来也脱不了身。毕竟，能冷落两个糙汉子，也不该冷落娇滴滴的姑娘。厚此薄彼，仁人志士所不取。
<br/>　　他不回来，程大雷和刘发财也好聊些性命攸关的事。
<br/>　　“程当家，您还是抓紧脱身吧。再不走怕就没机会了。”刘发财主动将自己和程大雷分开，正好也趁着这次机会从程大雷身边脱身。
<br/>　　程大雷轻轻点点头，拨弄着面前的茶杯。
<br/>　　这次入万卷山并不算毫无收获，至少已得到华氏一族的消息。况且明日的伏魔大会，就是为自己而设，饶是程大雷也不得不小心。
<br/>　　“程当家，您还要早作决定。您的名气是大的，明天来万卷山的高手也是多的。免不得就有人认得您的相貌，到时候要是被撞破，您要面对的，可就是整个东海的高手。”
<br/>　　“啊，你说得也不是没有道理啦。”程大雷点点头。
<br/>　　“但您好像不是很想走。”刘发财奇怪道。
<br/>　　程大雷再次点点头：“我想既然一味逃命，提心吊胆担心他们的追杀。倒不如留在此地，看他们凑在一起，会想出什么法子来对付我。”
<br/>　　“这个你要是被认出来？”
<br/>　　“嗳……天下之大，见过我相貌的又有几个。”程大雷摆摆手：“俗话说得好，越是危险的地方便越是安全，他们怎么会想到，伏魔大会要诛杀的魔头，就在大会上安稳的坐着呢。你说，我说得有没有道理。”
<br/>　　“这个，这个……”刘发财一时无语，却也不知道该佩服程大雷的艺高人胆大，还是说他脑袋有些拎不清。
<br/>　　到半夜时分，程大雷悄悄离开屋子，施展轻身功夫，将万卷山探了一遍。
<br/>　　明日万一有个闪失，不得已动起手来，自己及早探好地形，也方便跑路么。
<br/>　　而今夜，楚云生果然没再回来。
<br/>(本章完)
