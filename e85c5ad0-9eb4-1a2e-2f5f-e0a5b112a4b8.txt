　　第358章
<br/>　　“好姐姐，其实我也不愿来烦你，可他三番两次来扰我，我也是烦不胜烦呐。”
<br/>　　徐灵儿又卖弄自己的小聪明，却也算不得多高明，自然是被樊梨花一眼看穿了。樊梨花最是对这种小聪明嗤之以鼻，而樊梨花在徐灵儿眼里，自然是没脑子那种人，双方谁也看不上谁，也不怪关系不睦。
<br/>　　“依我看，他对你也不是没有心。若真无心，又何必求我来烦你。如今却也算不错，怎么也算是琴川之主，虽然人长得丑些……”
<br/>　　程大雷心中暗暗叹气，这徐灵儿不会说话呐，审美观也有问题，自己明明玉树临风，一表人才……怪不得事情做不成。
<br/>　　“不成就是不成，哪有那么多话讲，我已打定主意，终生不嫁，叫他，包括你别来烦我。”
<br/>　　徐灵儿一副快别扯淡的神情，然后道：“你们可真是难倒了媒人，好姐姐，究竟为何不成，你给我说一句，我也好去回他，让他死了这条心。”
<br/>　　“你去告诉他……”
<br/>　　程大雷呼吸便重，樊梨花究竟如何想得，程大雷一点也摸不着头脑，系统没给自己开这方面天赋呐。
<br/>　　樊梨花若有所思，眼神里流露出几许幽怨，忽地秀眉倒竖，冷声喝问。
<br/>　　“什么人！”
<br/>　　伴随着声音，手中的茶杯已经脱手，隔着窗户丢了出去，将窗户纸砸出一个大窟窿。
<br/>　　徐灵儿也是一惊，伸手便拔出腰间的短刀，眉间变历，和樊梨花一起窜出屋门。
<br/>　　屋外空荡荡的，一个人影都无，只有一个完整无缺的白瓷茶杯，搁在窗台上。
<br/>　　……
<br/>　　“都快一些，莫要偷懒，让本寨主看见偷懒的，绝饶不了你们。”
<br/>　　这一日上午，程大雷叉着腰，大声呵斥着干活的工人。
<br/>　　正干活的大家有些不明所以，向来程大雷还算是蛮和气的，也很少在工地上露面。再说如今天气秋高气爽，夏天太冷，冬天太热，现在正是合适干活的季节，所以大家也都没偷懒。
<br/>　　“呵，忙着呢？”樊梨花在程大雷身边出现。
<br/>　　“啊……”程大雷顿时紧张起来，这算是樊梨花第一次主动和程大雷说话，他道：“有事么，咱们去大厅说话。”
<br/>　　和珅说得不错，毕竟自己现在也是有身份的人，要讲究点上位着的体面。假若正被樊梨花当着众人训斥一番，自己难道不要面子么。
<br/>　　“用不着，送你一件东西。”
<br/>　　樊梨花将一样东西塞到程大雷手中，程大雷一怔，正是一个白瓷茶杯。
<br/>　　在他还没回过神来时，樊梨花已经走远了。
<br/>　　“唉……”
<br/>　　“喜欢就留着吧。”
<br/>　　樊梨花头也没回的甩出一句话。
<br/>　　程大雷本来想趁着施工的这段时间，把和樊梨花的事情理一理，可天不遂人愿，工程是一天比一天加快，但和樊梨花之间的事情却没有任何进展。
<br/>　　两人之间像是隔着一团乱糟糟的线，程大雷左思右想，找不到线头。
<br/>　　到后期，程大雷已经不断将人送回去，有意拖慢工程进度，可到了这一日，落玉寨所要做的任何改造已全部完成。
<br/>　　如今的落玉寨有整齐的民居，仓库、马场、聚义厅等建筑都做了改造。重点是外围的防御建筑，也都布置上了。
<br/>　　已经按照系统要求，建好所有需要的建筑，可程大雷并没有收到系统提示。
<br/>　　这让程大雷有些困惑，他再三确认，才发现了问题所在：如今的落玉寨缺少一个顶级人物。
<br/>　　一级到二级算是容易的，二级到三级其实也不困难，但三级到四级，却要迈过很大一个坎儿。所以动手抢，然后抢回来继续改造更现实一些。
<br/>　　但三级山寨需要的不仅仅是人口、建筑，还包括首领作用的顶级人物。
<br/>　　程大雷正有打算再通过点将台兑换一个顶级人物，目前落玉寨缺少的是主管山寨事物的管理人才，自己的琴川关有了和珅，但落玉寨还缺少这方面的人才。
<br/>　　他回到屋里，将门关了，唤醒脑海中的系统界面，决定兑换。
<br/>　　『拥有四级山寨，可以选择英雄职业，职业方向：生活职业，战斗职业』
<br/>　　随着程大雷拥有四级山寨琴川关，不仅抽取奖励时可以选择定向抽取，点将台也多了这方面功能。
<br/>　　程大雷翻阅着系统介绍，发现战斗职业包括：游侠、山贼、武将、恶霸等。其实却也是大同小异，有些适合战场杀敌，有些适合武林争斗，却也差距不大。
<br/>　　而相比之下，生活职业的划分要更详细些，包括：铁匠、马夫、医者、文官、军师，甚至还有和尚、道士。
<br/>　　程大雷选择了兑换生活职业，用一百万恐惧值兑换一个顶级英雄。好久没打过仗了，不打仗程大雷就没办法收获恐惧值，现在程大雷的恐惧值不多，必须省着点用。
<br/>　　随着程大雷的选择，繁杂的选项消失，只有点将台中的篝火亮起，星光汇聚，降落在其上。
<br/>　　嘟，恭喜获得顶级太监……高力士。
<br/>　　程大雷一下睁大眼睛，顿时有种无话可说的感觉。他心里明白，这是那倒霉的邪恶值有发挥作用了。
<br/>　　系统已经说得很明白，英雄相惜，臭味相投，自己是什么人，就会吸引到什么人来到什么。
<br/>　　以前自己『一个好人』属性还在的时候，有多少邪恶值一样被清零，所以能抽到刘关张这样的人，可遇见的未来，身边聚的一定是岳武穆、文天祥等等这样的忠臣良将，但是……自己的隐藏属性已经被自己搞没了。
<br/>　　程大雷悲催的预见到一个未来：自己身边聚集着秦桧、高俅、宇文化及等等这样臭名昭著的历史反派。
<br/>　　当然，因为有首领光环的存在，程大雷并不是无法限制他们，引导他们的负面buff去做好事，就像和珅一样。
<br/>　　但身边聚集这样一帮人，总觉得事情会变得越来越诡异。
<br/>　　程大雷脑海中突然冒出一副景象，一个剔着光头，背着盒子枪的男人大声道：队长，别开枪，我是正面角色啊。
<br/>　　回应他的是义正辞严的声音。
<br/>　　“我代表人民代表正义，枪毙了你”
<br/>
