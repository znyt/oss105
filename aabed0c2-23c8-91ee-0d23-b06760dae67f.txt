　　第10章
<br/>　　“……刚到蛤蟆岭，呼啦啦杀出一队人马，足有上百人，为首一人扛着一柄大斧，自称蛤蟆大王……”
<br/>　　黑石城，城主府。黄三元连夜赶回黑石城，正绘声绘色描述路上经历。他对面坐着一华服男人，一张圆脸，身材臃肿。
<br/>　　此人正是黑石城城主韩虎踞，他身后立着两名丫鬟，一着绿衣一着粉衣，看身量也就十六七岁，一般身高，一般模样，竟是一对罕见的双生子。
<br/>　　听着黄三元的话，韩虎踞头也没抬，眼皮依旧耷拉着。
<br/>　　“老黄心里怯了，脸上可不能怕，哗啦拔出腰刀，冲贼人大喊：想抢我们少夫人，就从我老黄尸体上跨过去……”
<br/>　　“……俺正要和贼子拼杀，又从身后杀出一队人，为首之人自称玉面书生徐神机……”
<br/>　　“咦！”韩虎踞身侧一白面中年男人道：“蛤蟆岭上有这么多山贼？”
<br/>　　“军师有所不知，二百人都不止，斜刺里又杀出一队人马，为首之人自号咏春叶问……”一路上，黄三元都在思索着如何应对，这番话在肚里滚了许久，最后将一人拆为三人，凑成三百之数：“老黄虽然拼死保护少夫人，可贼人数量太多，老黄落得一身伤，却还是让少夫人被贼人劫走……”
<br/>　　“老黄死罪呐，老黄死罪……”黄三元噗通跪倒，磕头如捣蒜：“老黄自知死罪，请城主与我一百兵丁，杀回蛤蟆岭，救出少夫人，再来城主面前领死……”
<br/>　　老黄一身是血，衣服上破烂不堪，此刻泪如雨下，身后随他从蛤蟆岭回来的兵丁也都跪在地上，唬得面如死灰，瑟瑟发抖。
<br/>　　“杀。”韩虎踞抬起头，口中吐出一个字。
<br/>　　“城主……”
<br/>　　地上跪着的人瞬间被押起来，拖着向大厅外面去，黄三元一张脸瞬间吓得惨白无比，已经撒下这弥天大谎，最后终究逃不过死路一条。
<br/>　　“慢！”
<br/>　　正这个时候，有一年轻人从大厅外走进，冲韩虎踞道：“爹爹，黄管家知道山贼情况，暂时还不能杀。”
<br/>　　随着黄三元回来的几人，差不多都被砍掉脑袋，人头滚落，鲜血淋漓，黄三元吓得魂都没了。
<br/>　　然后，他又被拖着回到大厅，只见韩虎踞身侧多了一个年轻人。面白无须，二十余岁上下，身上白衣纤尘不染，气质出众。
<br/>　　此人就是黑石城少城主韩玄之。
<br/>　　“黄管家，路上发生什么事，你再说一遍？”
<br/>　　黄三元一激灵，难道是自己刚才话里有什么破绽，韩玄之这是要当面揭穿自己。可这又有什么意义，反正逃不过一个死。
<br/>　　他哆哆嗦嗦又将刚才的话重复一遍，韩玄之听得是频频点头，眼角还有一丝隐隐的兴奋。
<br/>　　“爹爹，现在苏家小姐落在山贼手中，我请父亲虎符，让我带兵去蛤蟆岭剿匪，救出苏家小姐。”韩玄之声若金石。
<br/>　　“好孩子，那女人落在山贼手中，还有什么下场，不是被人侮辱，就是被人杀了。一个小娘皮死就死了，爹爹再给你找个好的，你看怎么样？”韩虎踞微倾身子，仔细询问。
<br/>　　“父亲！”韩玄之跪倒：“男儿若不能护住妻子，何以立足天地，请父亲准我带兵。”
<br/>　　“这么……”韩虎踞端起酒杯：“你要多少兵？”
<br/>　　韩玄之起身，口中吐出两个字：“两千。”
<br/>　　噗！
<br/>　　韩虎踞刚喝的一口酒喷了出来，大厅内其他几人也都睁大眼睛，满脸匪夷所思。
<br/>　　“公子，你知不知道两千人是什么概念？”那白面中年人几乎是哆嗦着问：“公子你有没有想过，两千人马的粮草是多大消耗，运送这些粮草又需多少车队，人员安排，行军路线……最重要的是，山贼可只有三百人呐……至于嘛？”
<br/>　　“这些我当然都有考虑。”韩玄之镇定道：“我只是比较喜欢碾压。”
<br/>　　白面中年人无语。
<br/>　　“黑石城一共才五千兵马，你这是要拿走爹一半人马啊，不，两千绝对不行，我最多给你一千人。”
<br/>　　“一千就一千吧，虽然有些少。”韩玄之遗憾道。
<br/>　　“我知道公子有领军的理想，不知公子这次打算如何攻打？”白面中年人问。
<br/>　　“我刚才已经考虑清楚了。”
<br/>　　“这么快就计划好了，我的儿子还真是棒呐。”韩虎踞满意道：“快说说……”
<br/>　　“我要请轻骑兵二百，弓兵二百，大刀兵三百，重甲兵一百，枪兵二百，凑齐一千之数，骑兵可奔袭，弓兵远程射杀，重甲兵能挡住贼人的冲锋，枪兵在重甲兵之后击杀，刀兵杀戮余孽。”韩玄之侃侃而谈：“蛤蟆岭距离此地一百五十里，大概需要一天一夜，长途奔袭，能缩短半日时间，到时攻敌不备，定能将贼人一网打尽。”
<br/>　　韩虎踞以掌击案，道：“不愧是我的儿子，既然你有这领兵的梦，这次就去试试手吧。”
<br/>　　黄三元早就吓懵了，总听人说少城主从小便修习兵法，最是聪明那样跃马扬鞭的古代名将，听他一番话侃侃而谈，可黄三元总有一种『你说得好有道理，我不知道怎么反驳，但总觉得哪里有些不对』的感觉。
<br/>　　而且……黄三元猛地激灵一下，山贼不过两人，真如果公子意气风发，杀到蛤蟆岭……黄三元浑身白毛汗，心想也许被砍头也是种不错的死法。
<br/>　　“黄三元，你随公子剿匪，将功折罪。”
<br/>　　“是，谢城主大恩大德，谢城主……”
<br/>　　韩玄之意气风发的拿着虎符走了，看着他脸上的精气神，怕是会彻夜不眠，安排这次行军计划。
<br/>　　大厅内，韩虎踞面向白面中年人：“薛军师，玄之的计划你觉得怎么样？”
<br/>　　这白面中年人名叫薛半川，他无奈摇摇头：“山林之中，骑兵的作用难以发挥，既要长途奔袭，那带重甲兵为何？况且土匪的山贼往往据险而守，奔袭对他们无效，少城主这计划不说漏洞百出，但也……不过，这些城主定也看出来了，却为何还是让少城主轻率行动？”
<br/>　　
<br/>(本章完)
